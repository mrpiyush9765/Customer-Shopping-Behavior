# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using Python, SQL, and Power BI. The goal is to identify customer purchasing patterns, sales trends, and business insights that can help improve decision-making.

## Objectives
- Analyze customer purchasing behavior.
- Identify sales trends and customer preferences.
- Perform data cleaning and preprocessing.
- Generate meaningful business insights.
- Create interactive dashboards and reports.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- SQL
- Power BI
- Jupyter Notebook

## Dataset
The dataset contains customer shopping information, including:
- Customer Details
- Product Information
- Purchase Amount
- Category
- Payment Method
- Purchase Date
- Shopping Behavior Metrics
